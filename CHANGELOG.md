# Changelog

All notable installer and distribution changes to DR Rene Magic Toolbox are documented in this file.

## [2.0.2] - 2026-07-17

### Added

- English Windows Installer wizard.
- User-selectable installation directory.
- DRRMT blue-and-gray installer branding.
- Matching multi-resolution Windows application icon.
- End-User License Agreement with a mandatory ZII-approved training-certificate requirement.
- Eventlog Puller for LGC and NGC

### Changed

- Improved installation progress and completion feedback.
- Improved readability of the welcome, completion, and installer banner screens.
- Updated the 64-bit installation layout for `Program Files`.

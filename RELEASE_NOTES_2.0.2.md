# DR Rene Magic Toolbox 2.0.2

This release introduces the redesigned DRRMT installer for 64-bit Windows.

## Highlights

- English installation wizard with a selectable destination folder.
- New blue-and-gray DRRMT branding and matching application icon.
- Clear installation progress and completion screens.
- Updated EULA requiring a valid ZII-approved training certificate for functional use.
- Eventlog Puller for LGC and NGC

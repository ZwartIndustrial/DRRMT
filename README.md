# DR Rene Magic Toolbox (DRRMT)

![DRRMT application icon](assets/DRRMT-icon.png)

DR Rene Magic Toolbox is Windows software published by Zwart Industrial Innovations (ZII).

## Download

Download the current installer from the [latest GitHub Release](https://github.com/ZwartIndustrial/DRRMT/releases/latest).

Current version: **2.0.2**

[Download DR Rene Magic Toolbox 2.0.2](https://github.com/ZwartIndustrial/DRRMT/releases/download/v2.0.2/DR_Rene_Magic_Toolbox_2.0.2.msi)

## Installation

1. Download the MSI installer from the Releases page.
2. Double-click the downloaded MSI file.
3. Read and accept the End-User License Agreement.
4. Select the installation directory.
5. Complete the installation wizard.

The installer is built for 64-bit Windows and contains the required .NET runtime.

## Training requirement

Functional use of DR Rene Magic Toolbox is permitted only by users holding a valid training certificate issued by ZII or by a training provider expressly authorised in writing by ZII. The complete terms are presented during installation and included in `EULA.rtf`.
